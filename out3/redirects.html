Redirects

As content gets changed over time, content moves. Cadmium provides a redirection filter to help you deal with this.

Defining Redirects

Redirects are defined in the META-INF/redirects.json file. Each entry in the JSON document defines a mapping from a regular expression for a path to a target URL. All redirects are returned with a status code of 301 Moved Permanently. Below is an example redirects.json:

[
  {
    "url":"http://www.meltmedia.com",
    "path":"/meltmedia"
  },
  {
    "url":"/redirects.html",
    "path":"/redirects/.*\.html"
  }
]

This will redirect all requests for /meltmedia to the meltmedia website. It will also redirect all requests to pages like /redirects/foo.html and /redirects/bar.html back to this page.

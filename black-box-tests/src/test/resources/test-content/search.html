Search

Cadmium comes with a Lucene indexer built in. Everytime content is uploaded, the index is updated, so the REST API will always server results for the currnent content.

The Search Endpoint

You can make a search request by doing a GET request to /api/search?query=QUERY_STRING. This will return you results as a JSON object:

{
  "number-hits": 2,
  "results":
  [
    {
      "path": "/path/to/file1.html",
      "score": 0.0112411,
      "title": "Title 1"
    },
    {
      "path": "/path/to/file2.html",
      "score": 0.0112411,
      "title": "Title 2"
    }
  ]
}

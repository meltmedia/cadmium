Email This Page

All Cadmium sites provide support for emailing the current page.

The Templates

Cadmium supports the Velocity templating engine for email templates. To define the templates for a welcome email, you define an HTML and a TXT template in /META-INF/email, like this:

./META-INF/email/email-this-page.html
./META-INF/email/email-this-page.txt

Email This Page emails can then be sent by passing the page path and email header information to the API.

Error Pages

Cadmium provides support for custom 400 and 500 level error pages. You can provide both specific and general error pages.

Specific Error Pages

To provide an error page for a specific status code, simple create an HTML file for that status code in the root of your content. For example, to create a 404 error page, you would add 404.html to your content.

General Error Pages

If Cadmium cannot find a specified error page, it will start looking for more general error pages. It does this by replacing the least significant digits with x's, until it finds a page to render. For example, if a 501 error occurs, Cadmium will look for:

  1. 501.html
  2. 50x.html
  3. 5xx.html

If none of those are found, Cadmium will fall back to an error page with instructions for configuring error pages.
